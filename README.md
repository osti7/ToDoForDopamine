# ToDoForDopamine

A minimalist, dark-themed habit and task tracking dashboard designed to visualize progress and trigger dopamine release upon task completion. Built with a focus on clean UI/UX and smooth animations.

---

## Features
* **Dynamic Task Calculation:** Automatically calculates progress steps based on the number of inputs or custom task numbers.
* **Minimalist UI:** Sleek black and vivid purple (`#4500ff` to `#cc00ff`) color palette designed for high focus.
* **Fluid Animations:** Uses cubic-bezier transitions for a satisfying, dopamine-inducing progress bar filling effect.
* **Zero Dependencies:** Built entirely with vanilla HTML5, modern CSS3 (Custom Variables/Flexbox), and pure JavaScript.

---

## How It Works (The Process)
1. **Setup:** Define your daily target by entering a number or typing your tasks line-by-line into the text area.
2. **Dynamic Step Scaling:** The JavaScript engine splits the text inputs, filters empty lines, and dynamically calculates the completion percentage per task (e.g., 5 tasks = +20% per task).
3. **Completion:** Click "Complete Task" to witness the progress bar smoothly advance with a glowing neon effect.

---

## Tech Stack
* **Frontend:** HTML5, CSS3 (Flexbox & CSS Variables)
* **Logic:** Vanilla JavaScript (DOM Manipulation & Event Listeners)
* **Design & Concepts:** Crafted with an "AI-Assisted Development" workflow using Cursor.
